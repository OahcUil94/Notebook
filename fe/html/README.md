# html相关知识点的整理

Google对a标签中的rel属性增加了新的搜索引擎说明符号：

- rel="sponsored" 用于赞助商链接
- rel="ugc" 用于用户提供的连接
- rel="nofollow" 则是是告诉搜索引擎完全忽略该链接
